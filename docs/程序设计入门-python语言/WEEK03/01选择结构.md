### 选择结构

#### 1.程序流程图

以简单的图形符号来表示问题的解决步骤，亦称**框图**。流程图是问题求解的最基本，最重要的分析技术。

#### 2.if 语句

语法：

```python
# if
if True:
    doSomethin()
otherThing()

# 多分支结构（chained）
# if-else
if True:
    doSomethin()
else:
    doAn0therThing()
otherThing()

# if-elif-else
age=80
if age>90:
    doSomethin(1)
elif age>60:
    doSomethin(2)
elif age>30:
    doSomethin(3)
else:
    doSomethin(4)

# 嵌套结构（nested）
if score >= 60:
    if gender == '女':
        print('Yes')

if score >= 60 and gender == '女':
    print('Yes')
```
