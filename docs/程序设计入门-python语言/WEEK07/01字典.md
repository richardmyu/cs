## 字典

一些列“键-值（key-value）”对，通过“键”查找对应的“值”。

创建字典要使用 `{}`：

```
my_dict = {'a': 123, 'b': 456, 'c_class': 789}
```

- 键必须是不可变的且不重复；
- 值可以是任何类型

> 字典中的键值对不以任何方式排序

### 1.字典方法

##### 1.1.访问字典

使用 `[]` 运算符，以键名作为索引：

```python
print(my_dict['a']) # study
```

##### 1.2.len

```python
print(len(my_dict)) # 3
```

##### 1.3.in

```python
print('b' in my_dict) # True
print('B' in my_dict) # False
```

##### 1.4.for

```python
for d in my_dict:
  print(d, my_dict[d])

# a study
# b work
# c eat and drink
```

##### 1.5.items/keys/values

```python
print(my_dict.items())
# dict_items([('a', 'study'), ('b', 'work'), ('c', 'eat and drink')])

print(my_dict.keys())
# dict_keys(['a', 'b', 'c'])

print(my_dict.values())
# dict_values(['study', 'work', 'eat and drink'])
```

##### 1.6.clear

```python
print(my_dict.clear()) # None
print(my_dict) # {}
```

##### 1.7.del

```python
del my_dict['c']
print(my_dict)
# {'a': 'study', 'b': 'work'}
```
