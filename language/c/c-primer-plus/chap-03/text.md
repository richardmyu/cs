| --------------------- | -------------------- | -------------------- |
| `int`                 | `signed`             | `_Bool`              |
| `long`                | `void`               | `_Complex`           |
| `short`               | &nbsp;               | `_Imaginary`         |
| `unsigned`            | &nbsp;               | &nbsp;               |
| `char`                | &nbsp;               | &nbsp;               |
| `float`               | &nbsp;               | &nbsp;               |
| `doublee`             | &nbsp;               | &nbsp;               |

通过这些关键字创建的类型，按计算机的储存方式可分为两大基本类型：整数类型和浮点数类型。
